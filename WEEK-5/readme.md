# Week-5: Front-End Development

This week focuses on front-end web development technologies, covering HTML, React, and Tailwind CSS.

## Topics Covered

### HTML
- Basic HTML structure and elements
- Nested lists and navigation
- Image handling and embedding
- Cloning and templating
- Profile cards and UI components
- Tables and data presentation
- Forms and user input
- CSS positioning properties
- Employee cards and dynamic content
- Blog spot layout
- Media queries for responsive design
- CSS Grid layout
- Tailwind CSS integration demo

### React
- Setting up a React project with Vite
- Component-based architecture
- JSX syntax and rendering
- State management basics
- Building interactive user interfaces

### Tailwind CSS
- Utility-first CSS framework
- Responsive design utilities
- Custom styling with Tailwind
- Integration with HTML projects

## Project Structure

- `html/`: Collection of HTML demos and examples
- `React/`: React application with Vite setup
- `Tailwind/`: Tailwind CSS project with custom configuration

Each subdirectory contains practical examples and implementations of the concepts learned during the week.
